## Virtual Environment

Create
```bash
python -m venv venv
```

Activate
```bash
./venv/scripts/activate.ps1
```

## Installation

Lembre-se de criar um ambiente python com as lib `env` ou `venv`. Um arquivo requiriments.txt está disponível com todas as libs utilizadas no projeto.

```bash
pip install -r requirements.txt
```
